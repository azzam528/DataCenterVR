\# Simulasi Data Center VR



Project VR simulasi pemasangan server, switch, konfigurasi IP, dan koneksi kabel jaringan menggunakan Unity dan XR Interaction Toolkit.



\## Fitur

\- Grab \& Snap Server/Switch ke rack

\- Koneksi kabel menggunakan controller VR

\- Kabel otomatis mengikuti posisi server/switch

\- Panel konfigurasi IP Address
